# 📧 Spam Email Detection System

## 📌 Overview
The Spam Email Detection System is a Machine Learning project that classifies emails into **Spam** or **Not Spam (Ham)**. It helps users filter unwanted messages and improves email security and productivity.

---

## 🚀 Features
- Classifies emails as Spam or Ham
- Uses Machine Learning algorithms
- Performs text preprocessing and cleaning
- Fast and accurate predictions
- Easy to use and implement

---

## 🛠️ Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Natural Language Processing (NLP)
